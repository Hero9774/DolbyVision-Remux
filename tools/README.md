# tools/

Dieser Ordner enthält optionale Drittanbieter-Tools.

## dovi_tool (optional)

Wird benötigt für die automatische Konvertierung von **Dolby Vision Profil 5 → Profil 8**
(behebt den Blaustich-Fehler auf Geräten ohne nativen DV-Decoder).

**Download:** https://github.com/quietvoid/dovi_tool/releases

**Installation:** `dovi_tool.exe` (Windows) in diesen Ordner (`tools/`) ablegen.

Das Hauptscript erkennt automatisch ob dovi_tool vorhanden ist und zeigt den Status in der GUI an.
